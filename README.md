#  Textbook AI: Chat with your PDF 

> **"Turning static textbooks into interactive mentors using RAG."**

---

###  The "Why"
Traditional textbooks are "one-way" conversations. You read, but you can't ask questions. **Textbook AI** bridges this gap by using a **Retrieval-Augmented Generation (RAG)** pipeline. 

This project was built to help students:
*  **Find** answers instantly without scrolling through 500 pages.
*  **Understand** complex concepts by asking the AI for simple analogies.
*  **Summarize** dense chapters into 5-minute study guides.
